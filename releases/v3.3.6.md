# Release Notes: v3.3.6

Welcome to **v3.3.6**! This release focuses on project topology flexibility and a modernized settings infrastructure, providing a more robust core for local database management.

## 🚀 Key Improvements

### 🌐 Dynamic Environment Management
The primary focus of this release is the move away from the hardcoded `DEV`, `STAGE`, `UAT`, `PROD` lifecycle.
- **Master Environments**: You can now define your own environment list globally and inherit them into your projects.
- **Custom Topology**: Add, rename, or reorder environments to match your organization's infrastructure.
- **Enhanced Mapping**: Improved logic for mapping sync pairs across your custom environments.

### ⚙️ Modernized Settings Architecture
We've rebuilt the settings interface from the ground up for better usability and a premium aesthetic.
- **Redesigned Sidebar**: Better categorization of global vs. project-specific settings.
- **Glassmorphism UI**: Refined visual styles using modern glass-blur effects and updated typography.
- **Responsive Layout**: Optimized for various screen sizes with improved navigation flow.

### 🛠️ Stable Core & Performance
- **Synchronized Monorepo**: Versions across `@the-andb/core`, `@the-andb/cli`, and the desktop application are now perfectly aligned.
- **IPC Reliability**: Fixed several race conditions in the inter-process communication between the UI and the database engine.
- **Internationalization (i18n)**: Updated translations for both English and Vietnamese locales to support the new environment management system.

## 📦 What's Included

- **Monorepo Version**: `3.3.6`
- **Modules**: `andb-desktop`, `andb-core`, `andb-cli`, `andb-mcp`, `andb-www`.
