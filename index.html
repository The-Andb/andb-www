MacOS "App is damaged" fix

Seeing a "damaged" error? This is a common macOS Gatekeeper protection for non-notarized apps. Being Open Source, our code is fully transparent and auditable — you can verify the safety of this command on our GitHub.

To fix it, open Terminal.app and run:

xattr -cr /Applications/The\ Andb.app

This removes the quarantine attribute and allows the app to launch normally.

The Vision

"Every deployment feels risky.
I wanted to change that."

After years as a backend developer, I kept hitting the same wall: databases are hard to version. Manually comparing schemas, missing columns, changed data types, silent production breaks... the risks only grow as your project evolves.

I realized that while code has Git and CI/CD, the database side was still manual and fragile. I asked myself: "Can I build a version control tool for DDL, like Git does for code?"

What started as a small CLI tool to solve my own pains has evolved into The Andb. It’s a developer-first tool built to restore trust and visibility in your migrations.

Designed for Reliability

The Andb ecosystem is built with a modular "Engine-UI-CLI" architecture, ensuring stability and speed regardless of how you prefer to work.

andb-core

The universal engine. Handles database connectors, atomic comparison logic, and script generation. Pluggable DataStore for both Disk and Memory.

andb-cli

A thin command-line wrapper for speed and automation. Perfect for CI/CD pipelines, staging syncs, and headless environments.

Getting Started Guide

Follow these simple steps to set up your environment and start migrating databases like a pro.

1

Bypass macOS Gatekeeper

Since The Andb is built by an independent developer, macOS Gatekeeper might block it initially. This is a standard security protocol.

Run in Terminal:

xattr -cr /Applications/The\ Andb.app
Terminal Fix Screenshot
Connections Screenshot
2

Configure Connections

Add your source and target databases in the Settings. Currently supporting MySQL/MariaDB with PostgreSQL coming soon.

  • Support for Host, Port, SSH Tunnels, and SSL
  • Built-in connection tester
3

Sync & Migrate

The magic happens here. Select your environment pair and compare schemas instantly. Review SQL changes before applying.
