# v3.3.4 - Branding Excellence & Visual Clarity

This release focuses on brand authenticity and a high-impact redesign of the core comparison workflow to provide a world-class experience.

## What's New

### 🛠️ High-Impact Comparison Header
- **Database Node Redesign**: The connection header now uses a high-contrast "Node" structure with dedicated `Database` icons, making it instantly clear which two environments are being compared.
- **SOURCE & TARGET Badges**: Added bold, colored badges to provide an immediate visual anchor for data flow direction (Blue for Source, Emerald for Target).
- **Data Pipeline Bridge**: Replaced the minimalist arrow with a structured synchronization "Pipeline" visualization.

### 🎨 Branding & Aesthetic Polish
- **Logo Authenticity**: Restored the original `andb-logo-transparent.png` across the website and desktop app.
- **Minimalist Cleanup**: Removed all redundant boxes, faint borders, and background layers from the navigation and comparison toolbars to achieve a modern, floating aesthetic.
- **Typography Refinements**: Standardized high-density tracking and weights across all headers for a more professional finish.

### 🔄 Monorepo Synchronization
- **Version Bump**: All monorepo packages (`core`, `cli`, `desktop`, `mcp`, `landing`, `test`) have been synchronized to **v3.3.4**.
- **Architectural Alignment**: Ensured full compatibility between the dynamic IPC storage layer and the new minimalist UI components.
