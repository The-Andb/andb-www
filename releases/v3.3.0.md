# 🚀 TheAndb v3.3.0

## 🌟 Multi-Tab Experience

- **Chrome-inspired Tab Bar**: New multi-tab interface for Schema, Compare, and History with flexible shrinking and horizontal scrolling.
- **Independent State**: Isolated data loading for each tab—no more stuck spinners when switching between tasks.
- **Scrollable Header**: Automatically scrolls the active tab into view, optimized for high-intensity multi-window workflows.

## ⌨️ Centralized Keyboard Shortcuts

- **Power User Flow**: Centralized shortcut system for lightning-fast navigation (`Cmd + W` to close tabs, `Cmd + [` / `Cmd + ]` to switch tabs).
- **Shortcuts Modal**: Modern KBD-style shortcuts guide accessible via `Cmd + /`.
- **Global Commands**: Added support for `Cmd + R` (Refresh View) and `Cmd + F` (Quick Search Focus).

## 🔄 Auto-Update & "What's New"

- **ApplicationUpdater**: Automated background update checks and notifications on startup.
- **Changelog Reveal**: "What's New" modal automatically appears after upgrades so you never miss a feature.
- **Stability Fix**: Resolved `EPIPE` crashes in packaged applications for a more reliable production experience.

## 🎨 Branding & Identity

- **UI Refinements**: Polished modals and transition effects for a more premium, fluid feel.

## 🛡️ Core Engine (v4.0.3)

- **Log Isolation**: Shielded system log streams to prevent pipe conflicts in production environments.
- **Resource Management**: Optimized memory footprint when handling massive DDL objects across multiple active tabs.
