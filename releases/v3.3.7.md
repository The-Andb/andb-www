# Release Notes: v3.3.7

Welcome to **v3.3.7**! This release introduces initial AI agent support and significantly bolsters our UI stability through automated regression testing.

## 🚀 Key Improvements

### 🧠 Initial AI Integration
We are laying the foundation for intelligent database assistance.
- **Agent Framework**: Initial integration of AI agents and support infrastructure.
- **Automated Workflows**: New internal agents to assist with release management and maintenance.

### 🎨 UI Stability & Themes
Ensuring a consistent look across all 13 built-in themes.
- **Theme Regression Testing**: Automated visual testing suite to prevent UI regressions across our extensive color palette.
- **Glassmorphism Refinement**: Minor aesthetic tweaks to the project settings and selection interfaces.

### 🛠️ Monorepo Orchestration
Improving the developer experience for contributors and maintainers.
- **Unified Syncing**: New scripts for synchronized branch switching across the entire monorepo.
- **Workspace Management**: Smoother handling of local package dependencies and workspace references.

### ⚙️ Minor Fixes & Refinements
- **Settings Modernization**: UI refinements for project settings and picker.
- **Build Optimization**: Updated several core dependencies for better performance and security.

## 📦 What's Included

- **Monorepo Version**: `3.3.7`
- **Modules**: `andb-desktop`, `andb-core`, `andb-cli`, `andb-mcp`, `andb-www`.

*For those exploring the beta AI capabilities: Use the new internal agents to streamline your development workflow. Development continues on the AI support branch.*
