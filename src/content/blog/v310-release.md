# 🚀 The Andb v3.1.0 Stable Release: Master the DDL

We are thrilled to announce the first major STABLE RELEASE of **The Andb** – the simplest database migration tool. This version marks a significant milestone in our journey, bringing a refined user experience, robust features, and a fresh new brand identity.

🌟 **Major Highlights**

### 🔍 Search Everything
Locate any string within your database procedures, functions, and views instantly. The global search bar allows you to find definitions and logic across your entire project schema with ease.

![Search Everything Placeholder](https://via.placeholder.com/800x450?text=GIF:+Search+Everything+Demo)

### 🎯 Go to Definition
Navigation is now effortless. In the DDL viewer, simply use **Cmd+Click (Mac)** or **Ctrl+Click (Windows)** on any identifier to jump directly to its definition. No more manual searching!

![Go To Definition Placeholder](https://via.placeholder.com/800x450?text=GIF:+Go+To+Definition+Demo)

### ↔️ Resizable Columns
We've heard your feedback! You can now drag to resize columns in the Table Detailed View, Connection List, and DDL Visualizer. Your preferred widths are remembered automatically.

![Resizable Columns Placeholder](https://via.placeholder.com/800x450?text=GIF:+Resizable+Columns+Demo)

### 📦 pnpm Monorepo
The entire monorepo has been standardized to `pnpm` for faster, more predictable installations and a cleaner environment.

### 💾 Custom SQLite Path
You can now relocate your internal metadata database to a custom folder (like Google Drive or Dropbox) to ensure your projects are always backed up.
