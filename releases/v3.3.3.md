# v3.3.3 - Stability & UX Refinement

This release focuses on platform stability, integration test resilience, and significant UI/UX improvements across the connection and comparison workflows.

## What's New

### 🛠️ Platform Stabilization
- **Integration Test Resilience**: Resolved critical `ts-jest` path resolution issues and modernized the testing infrastructure with ESM/ES2020 support.
- **Core Module Fixes**: Standardized `better-sqlite3` imports across the monorepo to ensure compatibility between Core, CLI, and Desktop environments.
- **Dependency Alignment**: All monorepo packages bumped to v3.3.3 to ensure full architectural synchronization.

### 🎨 UI/UX Excellence
- **Compact Inline Confirmation**: Replaced intrusive browser `alert()` and `confirm()` calls with a custom, premium slide-out inline confirmation UI for all delete actions.
- **Enhanced Visual Hierarchy**: Standardized UI elements to 120% scale with a refined color palette (black primary text, white-on-crimson action buttons).
- **Intelligent Database Context**: Navigation and Compare sidebars now display the active database name for clearer multi-environment context.

### 🐛 Critical Bug Fixes
- **Connection Duplication**: Fixed a synchronization bug in the IPC storage layer that caused duplicate entries when importing from Global collections.
- **Environment Selection**: Standardized environment management to a robust single-select model, preventing conflicting multi-environment states.
