# 🚀 TheAndb v4.0.1 Release

## ✨ Major Features
- **Pair Environment Mirroring**: Compare View now displays both Source (SRC) and Target (TGT) database tree structures under the selected pair.
- **Environment Toggle**: Clicking on environment name labels (e.g. `DEV → UAT`) under the pair card now toggles the visibility (completely shows/hides) of the corresponding tree.
- **Checkbox Support**: Restored database and category exclusion checkbox visibility in the sidebar across all tree explorer tabs.
- **Custom Table Layouts**: Added settings selectors for Font size (Small, Medium, Large) and Row line height (Small, Medium, Large) in the Data inspector tab.
- **Auto-Fit & Scrolling**: Integrated horizontal scrolling support and automatic column auto-fit sizing upon query execution in the Table data viewer.
- **Header Column Filters**: Added toggleable column header filters with mathematical comparison operator support (`=`, `>`, `<`, `<=`, `>=`) for numeric values.
- **Hybrid Comparison Engine**: Added aggressive routine normalization (`normalizeRoutineDDL`) in `ParserService` to ignore whitespace, comments, backticks, and formatting for procedures, functions, triggers, and views.
- **Metadata Synchronization**: Integrated schema/DDL character set and collation extraction and tracking (saved to SQLite `ddl_exports` and matched in `compareFromStorage`).
- **Seamless AI Diagnosis**: AI Diagnose button in Database Pulse Monitor now opens and streams findings directly in the main AI Assistant Panel conversation.

## 🛠 Fixes & Improvements
- Resolved comparison query execution loading state triggers and improved empty view descriptions.
- Cleaned up duplicate localization string mappings in the Vietnamese (`vi.yaml`) translation file.
- **MySQL Routine Syntax Errors**: Corrected `cleanDefiner` to normalize spaces using `/[ \t]{2,}/g` instead of `/\s{2,}/g`, preserving newlines so that trailing parameter comments do not comment out `BEGIN` and subsequent declarations.
- **Git-Sync Path Error**: Added options fallback for target database names to resolve `TypeError: path must be a string` in `SchemaMirrorService` when source connection details are empty.
- **Batch Migrate UI Refresh**: Solved case-sensitivity mismatch (`PROCEDURES` vs `procedures`) to ensure the results map updates correctly and removes completed items from the diff list.
